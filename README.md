### About

Software engineer with a mathematics background. I work on accessibility, building tools that find issues in component libraries before they reach production. Most of my time goes into building software for inclusion: a11y tooling and apps for neurodivergent kids.

Trilingual (NL · DE · EN). Outside work: field hockey, go-karts, astronomy.
